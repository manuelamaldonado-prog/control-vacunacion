💾 Backup, Portabilidad y Guardado Permanente

¿Cómo funciona el guardado?

• Los cambios se guardan automáticamente en este navegador.

• Para mayor seguridad, usa el botón azul de abajo para descargar un archivo de respaldo (Backup).

• Importante: Este archivo es lo que necesitas para llevar tus datos a otra computadora.

1. GUARDAR / EXPORTAR

Presiona aquí para descargar una copia de seguridad en tu carpeta de Descargas.

2. RECUPERAR / IMPORTAR

Usa esto solo si quieres cargar datos desde un archivo guardado anteriormente.

⚠️ ¡CUIDADO! Esto borrará los datos actuales y los reemplazará por los del archivo.
